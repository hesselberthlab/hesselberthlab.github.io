    Problem Set Submission¶§

    Specify the root of your project directory and create a tar file of the whole directory like this (change STUDENTID to your student ID):

    $ projectdir=$HOME/project
$ tar -cvf STUDENTID-problem-set.tar $projectdir

    Upload the tar file to the Problem Set at the Canvas site [1] to complete the submission.

    [1]https://ucdenver.instructure.com/courses/11079/assignments/syllabus
