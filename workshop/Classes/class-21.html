Class 21 : R : DataFrames & Plotting¶§

Goals¶§

  1. Understand data.frames in more detail
  2. Practice manipulating data.frames
  3. dplyr, ggplot

data.frames¶§

Read in a data.frame, view the first few lines and extract a single column

covs = read.delim('expr-geno-covs.txt')
head(covs)
head(covs$genotype)

# or
head(covs[,"genotype"])

To extract multiple selected columns:

head(covs[,c("genotype", "expression")])

Extracting From data.frames¶§

The order is rows, columns. By having nothing before the comma:

head(covs[,c("genotype", "expression")])

We extract all rows. We can manually extract the first 5 rows and our 2 columns as:

covs[1:5, c("genotype", "expression")]

Named rows in data.frames¶§

In expr-geno-covs.txt, there is a sample column, we may wish to use that as the row id:

covs = read.delim('expr-geno-covs.txt', row.names='sample_id')
head(covs)

covs[c('sample_1', 'sample_14'),]

We can still extract them with numbers:

covs[c(1, 4),]

Warning

Be careful with data.frames where the row.names are from an integer column

Boolean Operations on Colums¶§

is_old = covs$age > 65

now is_old is a list of TRUE / FALSE values. We can extract only those > 65 as:

old = covs[is_old,]
# same as:
old = covs[covs$age > 65,]

We can combine selections with ‘&’ for and and ‘|’ for or

old_with_disease = (covs$age > 65) & (covs$condition == "case")
owd = covs[old_with_disease,]

# with subset
old <- subset(covs, age > 65)

# select on membership
genos <- c('AC', 'CA')
hets  <- subset(covs, genotype %in% genos)

Excercises¶§

Remember for combining expressions, you can create a variable for each, is_AA, is_CC and then combine after.

  1. How many people have genotype ‘CC’
  2. How many people have genotype ‘CC’ or ‘AA’?
  3. How many people have genotype of ‘CC’ or ‘AA’ and are under 65 years old.
  4. How many males have genotype of ‘CC’ or ‘AA’ and are under 65 years old.

dplyr review¶§

dplyr provides these simple methods:

  1. summarise()
  2. filter()
  3. select()
  4. mutate()
  5. arrange()
  6. group_by()

dplyr¶§

dplyr also provides an operator called %.% that allows you to chain manipulations together.

To get mean expression level by condition (case/control)

covs %.% group_by(condition) \
     %.% summarize(count=n(), mean.expr=mean(expression))

Mean expression by condition and genotype

covs %.% group_by(condition, genotype) \
     %.% summarize(count=n(), mean.expr=mean(expression))

Exercise¶§

  1. What are the mean ages by cases and controls?
  2. What are the mean ages by genotype?
  3. How can you order the output by ascending and descending mean expression?

ggplot¶§

Above, we had mean expression by condition and genotype as:

covs %.% group_by(condition, genotype) \
     %.% summarize(count=n(), mean.expr=mean(expression)) \

We can add to that expression (after typing ‘library(ggplot2)’)

%.% ggplot(aes(x=genotype, y=expression) \
    + geom_histogram(stat='identity')

how can we change the color of all the bars to ‘red’? [Hint, it’s not color =’red’]

ggplot histograms¶§

Since expr-geno-covs.txt is already in long format, we can use it directly in ggplot:

ggplot(covs, aes(x=expression)) +
       geom_histogram() +
       scale_x_log10()

Exercise¶§
